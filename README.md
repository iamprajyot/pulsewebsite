# Pulse - Interval Notifier Landing Page

This repository contains the source code for the official landing page of **Pulse - Interval Notifier**, a beautifully designed interval notifier for iOS.

## 🚀 About Pulse

Pulse is a modern interval notifier crafted for iOS. Whether you're staying focused, keeping hydrated, or tracking your workouts, Pulse works seamlessly in the background and even when your screen is locked.

### Key Features
- **Modern Design**: A stunning, user-friendly interface with attention to every detail.
- **Background Alerts**: Works flawlessly when the app is closed or the device is locked.
- **Haptic Feedback**: Custom haptic patterns that work silently and seamlessly.
- **Pulse Pro**: Unlock 24+ premium notification sounds, set quiet hours, and schedule intervals perfectly.

## 🌐 Website Features

- **Responsive Design**: Optimized for mobile, tablet, and desktop viewing.
- **Dark Mode Aesthetics**: Sleek dark-themed UI following modern design principles.
- **Fast Performance**: Built with vanilla HTML and CSS for lightning-fast load times.
- **SEO Optimized**: Proper heading hierarchy and meta tags for search engine visibility.

## 🛠️ Tech Stack

- **Structure**: HTML5
- **Styling**: Vanilla CSS3
- **Typography**: [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- **Assets**: Custom logo and App Store branding
